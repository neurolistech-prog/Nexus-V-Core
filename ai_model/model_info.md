# AI Model Specifications
- **Model Type:** Convolutional Neural Network (CNN)
- **Framework:** TensorFlow / Keras
- **Input:** 224x224 RGB Images (Product Scans)
- **Classes:** Electronics, Clothing, Books, Home Goods.
- **Accuracy Target:** 94% on validation set.

*Note: The weights file (.h5) is too large for GitHub LFS in this prototype, but the architecture is ready for training on Google Cloud Vertex AI.*
